8.3: Buscar

Esta sección explica cómo fijar el texto que se muestra para las listas desplegables de la página de búsqueda. Bajo la pestaña "Formato", pulse "Búsqueda".

Este panel ofrece una tabla que enumera cada índice de búsqueda, nivel y partición. Aquí puede ingresar el texto a ser usado para el nombre del índice en la lista desplegable de la página de búsqueda. Este panel sólo permite fijar el texto en un idioma, el utilizado en ese momento por GLI. Para traducir el texto a otro idioma, use el panel Traducir Texto de la solapa Formato (Véase Traducción de textos).
