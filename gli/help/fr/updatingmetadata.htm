5.5: Modifier ou éliminer les métadonnées

Pour modifier ou éliminer un élément de métadonnées, sélectionnez d’abord le fichier approprié, puis la valeur de la métadonnée dans le tableau. Modifiez la valeur de ce champ et/ou supprimez tout le texte si vous désirez éliminer la métadonnée.

Le processus est identique pour la mise à jour d'un dossier avec des sous-dossiers ou plusieurs fichiers ; dans ce cas vous pouvez seulement mettre à jour les métadonnées qui sont communes à tous les fichiers/dossiers sélectionnés.

L' Arborescence de valeurs présente toutes les valeurs actuellement attribuées de même que les valeurs précédentes qui ont été attribuées pendant la présente session ; toutes les valeurs éliminées ou déjà modifiées seront donc toujours présentes dans l' arborescence. Fermer la collection et la rouvrir éliminera les valeurs qui ne sont plus attribuées.
