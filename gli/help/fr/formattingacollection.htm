8: Personnalisation de l'apparence de la collection

Une fois que vous avez construit la collection, vous devez décider de la manière dont elle doit apparaître à l'utilisateur. Quels noms doivent être utilisés pour la liste déroulante de l'index dans le formulaire de recherche? Comment les résultats de recherche doivent être affichés? Quelles métadonnées doivent être affichées lorsqu'un document est visualisé? Tout cela peut être personnalisé ; cette section explique comment le faire.
