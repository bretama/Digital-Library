9.3: Экспорт коллекций в другие форматы.

Greenstone может экспортировать содержимое и/или метаданные коллекции в несколько стандартных форматов, включая METS, Dspace и MARCXML.

Для экспорта одной или более коллекций откройте меню "Файл" выберите "Экспорт...". Можно выбрать формат экспорта, выбрав его из списка "Экспортировать в". Укажите имя каталога, куда вы хотите положить экспортированные файлы— файлы заканчиваются <путем к greenstone>/tmp/exported_xxx, где xxx – указанное имя. Выберите одну или более коллекцию, а потом кликните "Экспорт коллекции".

Есть некоторые другие опции, специфичные для различных форматов. Вы можете указать файл XSLT, который будет применен к результирующим XML-документам для настройки формата вывода. Экспорт в MARCXML использует файл отображения для сопоставления метаданных Greenstone полям MARC. Файл отображения по умолчанию сопоставляет только метаданные Dublin Core. Можно указать свой файл отображения вместо него.
