8.8: Метаданные Депозитора

Депозитор Гринстоун позволяет пользователю добавлять новые документы в существующие коллекции через веб интерфейс. Данный раздел описывает панель метаданных, где определяются поля описания новых документов в Депозиторе. Любые наборы метаданных ассоциированных с текущей коллекцией будут доступны для выбора. Если здесь нет других наборов метаданных назначенных "Набором извлеченных Гринстоун данных" ассоциированных с коллекцией. "Набор дублинского ядра" будет применен по умолчанию. Для изучения Депозитора, обращайтесь к онлайн руководству на английском по адресу http://wiki.greenstone.org/wiki/gsdoc/tutorial/en/depositor.htm. Нажмите вкладку "Format", нажмите "Depositor Metadata".

Панель метаданных Депозитора показывает список полей метаданных. Если имеется более одного набора метаданных в данной коллекции, соседние наборы метаданных будут представлены различными цветами. При помещении курсора сверху элемента метаданных: появится инструментальная подсказка.

Выделите те которые вы будете использовать для описания новых документов с помощью Депозитора. Выпадающий список с выбором двух вариантов появится сразу после выбора элемента. Это позволит указать тип ввода для элемента в веб интерфейсе. "текст" означает что будет использован ввод в одну строку, где "текстовая область" означает что будет применен многострочный ввод. Выберите соответствующий тип для каждого поля.

Как минимум один элемент метаданных должен быть выбран. Если имеется только один выбранный элемент в списке, отменив выбор вызовет появление сообщения: "Должен быть выбран хотя бы один элемент метаданных".
