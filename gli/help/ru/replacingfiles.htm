4.5: Переименование и замена файлов

Файлы можно переименовывать, щелкнув правой кнопкой по ним и выбрав «Переименовать» из списка. Введите новое имя и щелкните «ОК».

Файлы в коллекции можно заменять правым щелчком по файлу и выбрав «Заменить». Будет открыт просмотр файлов: переместитесь к новому документу и щелкните "Открыть". Новый документ заменит старый в коллекции, и все метаданные будут перемещены для него. Это полезно для замены документов-макетов настоящими документами.

Некоторые типы файлов преобразуются в HTML сторонним программным обеспечением при импорте, например, Word, Excel, PDF и т.д. HTML полученный при этом может быть не очень хорошо отформатирован. Эти документы имеют возможность усовершенствования правой кнопкой мыши: "Заменить HTML версией". Выбор этого варианта заменит HTML версию оригинального файла, новым, который может быть отредактирован.
