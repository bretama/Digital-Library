6.4.3: Заданные языки

В этом разделе детализировано,как ограничить индексы поиска специфическими языками. Вы совершаете это при регенерировании разделения,используя клавишу "Заданные языки" в перечне "Индексы разделения".

Языковые разделы используют метаданные, чтобы определить, какие документы имеются в указанных языках, и поэтому должны быть включены в раздел. Гринстоун генерирует "ex.Language" метаданные для большинства документов, по умолчанию это метаданные для использования. Однако, может быть изменено на другой элемент метаданных с помощью параметра "Метаданные языка:".

Чтобы добавить новый язык для разделения, используйте клавишу "Заданные языки" для построения индекса для этого. Выберите желательный язык из ниспадающего списка "Языки для добавления" и щелкните "Добавить язык"

Для изменения существующего раздела выберите его из списка "Присвоенные языковые разделы", измените выбранные языки в списке "Языки для добавления" ниже, и щелкните "Заменить раздел".

Чтобы удалить язык,выберите его из списка "Выборы языка" и щелкните "Удалить язык".

Порядок, в котором индексы указываются в списке «Присвоенные индексы» - это тот порядок, в котором они появляются на странице поиска. Используйте кнопки "Двигаться вверх" и "Двигаться вниз" для изменения этого порядка.

Чтобы установить язык по умолчанию,выберите его из списка и щелкните "Установка по умолчанию".

Имена, используемые для ниспадающего списка индексов на странице поиска могут быть указаны в "Поиск" части "Формат" панели, (см. Поиск).
